# Voyager

Voyager is a travel website developed as a web development project. It helps users explore travel destinations, view tour guides, and find travel information through a simple and responsive interface.

## Features

- Home page
- Destinations page
- Tour guides page
- Guide profile page
- Sign up page
- About Us page
- Contact Us page
- Responsive design

## Built With

- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
